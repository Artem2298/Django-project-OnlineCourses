# Online Courses Platform

## Overview

This is an online courses platform built with Django. The application allows users to register, login, view courses and lessons, and take tests. Instructors can create and manage courses and lessons.

## Features

- User Registration and Login
- Profile Management
- Course and Lesson Viewing
- Test Taking
- Instructor Course and Lesson Management

## Technologies Used

- Django
- Bootstrap

### Prerequisites

- Python 3
- Django 3
